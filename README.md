# Feeder

##  Data Scraper with Automation
+ `docker-compose up --build -d` to create a mongodb database

+ `cd ./server`
+ `npm install`
+ `npm run db:init`
+ `npm install -g pm2`
+ `pm2 start ecosystem.config.json`
